# sum-array-elements

## Overview
Wasm に対応した，配列の各要素の総和を計算するプログラム

### Option
```
-n N
```
スレッド数（配列の分割数）を指定する

### Command Format
+ Rust native
```

```
+ Wasmer + WASIX
```

```

## Requirement
+ Rust
+ Wasmer

### Install Wasmer

## Usage

## Features
### Flow
1. 配列を生成する
2. 配列を与えられたスレッド数に分割する
3. 分割した部分の総和を各スレッドが求める
4. 各スレッドの計算の終了を待ち，各和の総和を求める
